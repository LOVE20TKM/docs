# LOVE20 协议

LOVE20 是一个开源的去中心化社群铸币协议。旨在让每一个人能够将热爱铸造为代币。

[去中心化前端地址](https://love20tkm.github.io/interface/)

## 关于开源

本协议的所有文档、代码全部开源，并汇总于[Github](https://love20tkm.github.io/docs/)（即本页面）。
